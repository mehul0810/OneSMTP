# Docs Update Protocol

Every feature/fix PR must include a docs impact note.

## Required PR Section

- Feature summary
- User impact
- Settings/config changes
- Retry/failover behavior changes
- Data model/logging changes
- Upgrade or migration notes
- Docs files updated

## Merge Gate

PRs must either:

- Update relevant docs files, or
- Explicitly state "No Docs Impact" with reviewer approval.
